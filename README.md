# Эмулятор машины Тьюринга
Программа на С++ с использованием Qt для визуализации работы машины Тьюринга

## Возможности
* Задание алфавита и дополнительных символов
* Редактирование программы в таблице ('символ, R/L, состояние')
* Пошаговое и автоматическое выполнение
* Регулировка скорости
* Вихуализация ленты с головкой

## Управление
* **Задать строку** - помещает введённое слово на ленту
* **Запуск** - автоматическое выполнение
* **Шаг** - один такт
* **Сброс** - возврат к исходному слову и состоянию q0
* **Вставить Λ** - вставляет пустой символ в текущую ячейку таблицы
